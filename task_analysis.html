HTA Diagram

Notes:
- Main goal for users is to learn a technical skill
- 3 main tasks for users to accomplish: go to our website, get recommendations for tutorials, and complete a tutorial


Task 1 Subtasks

Notes:
- The first task (going to our website) has two subtasks
- The home page is where we introduce our site’s purpose and provide the user with the necessary information to proceed


Task 2 Subtasks

Notes:
- The second task is getting recommendations for tutorials
- The user will be provided “tracks” to choose from
- These “tracks” will identify whether the user:
    - Wants to learn a skill to supplement their career (ex: a specific programming language)
    - Wants to increase understanding of editing software for personal interest
       - (ex: develop Adobe skills for art/photography/videos/etc.)
    - Wants a way to share or develop a passion (ex: building a website based on hobby or side interest)
    - Already knows what kind of skill they want to learn, will be directed to our full list of tutorials


Task 3 Subtasks

Notes:
- The third task is to complete a tutorial
- Tutorials will vary depending on the type, but most will follow a basic structure:
    - Video and/or text instructions
    - An example for the user to follow along with
    - Practice problem(s)
    - Recommendations for other, more detailed resources if the user wants to develop skill further
